# Gerold — Portfolio (starter)

This is a one-page dark neon portfolio starter modeled after the provided design.

Quick start

1. Place your hero/profile image at `assets/images/profile.jpg` (recommended size ~ 600×600).
2. Open `index.html` in a browser.

What is included

- `index.html` — one-page layout with all major sections.
- `assets/css/style.css` — theme, glassmorphism, layout and responsive rules.
- `assets/js/main.js` — simple interactions (smooth scroll, services toggle, demo contact).

Customization

- Replace text, imagery and project thumbnails in `index.html`.
- Hook the contact form to a backend or a service like Formspree/Netlify Forms.
